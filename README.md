# vyges-ip-catalog

Auto-synced mirror of `vyges-metadata.json` files published in the
[`vyges-ip`](https://github.com/vyges-ip) GitHub organisation.

For discovery, search, and integration use the public service at
**https://catalog.services.vyges.com**.

Questions: **https://vyges.com/contact**.
